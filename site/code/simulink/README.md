# Simulink 教学示例

本目录提供高超声速控制教程的最小 Simulink 对照样例。内容先从最稳妥的动压监控开始，再给出一个教学级守护纵向闭环模型，把第 4 章的飞行走廊公式、第 10 章的闭环指标和 Simulink 日志接口连起来。

本目录的说明、指标和公式与正文保持同一套口径：数学表达直接用 $ 和 $$，普通术语不写成代码样式；如果某个符号在本说明中首次出现，应在就近位置把它讲清楚。

模型主要有两个：models/hgv_q_monitor.slx 负责计算动压并检查是否越过阈值，models/hgv_longitudinal_closed_loop.slx 负责运行教学级守护纵向闭环。前者先把约束检查说清楚，后者再把闭环链路说清楚。

动压监控模型计算动压：

$$
\bar{q}=\frac{1}{2}\rho_0\exp(-h/H)V^2
$$

其中，这条公式把动压、海拔和速度之间的关系先说明清楚，便于后面检查阈值。

并输出 $\bar{q}$、$m_q$ 和 $\Pi_q$ 三个量。$\bar{q}$ 表示动压，$m_q$ 表示动压安全裕度 $q_{\mathrm{limit}}-\bar{q}$，$\Pi_q$ 则表示是否满足 $\bar{q}\le q_{\mathrm{limit}}$。

纵向闭环模型是教学级守护纵向闭环，包含参考轨迹、动压参考治理、PID 风格速度高度控制、简化纵向动力学和执行机构状态，以及 $\mathrm{sim\_state}$、$\bar{q}$、$I_q$、参考量和命令量日志。

## 运行方式

建议按下面顺序执行：

1. 在 MATLAB 中运行动压监控检查脚本。
2. 再运行守护纵向闭环检查脚本。
3. 如果需要命令行入口，可分别用 matlab -batch 调用这两个脚本。

烟测会先验证动压监控模型：运行一次仿真，并断言 Simulink 输出的 $\bar{q}$ 与解析公式一致。随后再验证守护纵向闭环模型：运行闭环仿真，并断言状态、动压、参考和命令日志为有限值，同时检查 $h_{\mathrm{rms}}$、$V_{\mathrm{rms}}$、$q_{\max}$ 和 $I_q$。

已验证结果：

- **Simulink smoke test passed**：$\bar{q}=29081.837883767$, $m_q=40918.162116233$, $\Pi_q=1$
- **Simulink closed-loop smoke test passed**：$\Pi=1$, $h_{\mathrm{rms}}=723.112$, $V_{\mathrm{rms}}=366.780$, $q_{\max}=67059.569$, $I_q=0.000$

## 边界

这些模型是教学模型，不是高保真飞控模型，也不能用于工程定型。守护纵向闭环只覆盖最小纵向闭环，不等价于完整控制器族 Simulink 对照。后续应继续补齐参考治理器、控制器、执行机构、飞行动力学和指标统计子系统，并把 LQR、NDI、反步、滑模和屏障层的 Simulink 对照，以及与 MATLAB/Python 指标一致的闭环烟测继续接上。
