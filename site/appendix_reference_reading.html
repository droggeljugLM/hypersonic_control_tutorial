参考阅读与文献训练

本附录给出参考阅读路线、检索关键词、阅读记录模板和文献综述写法。它不是正式参考文献表，也不替代数据库检索。

阅读顺序

建议先读飞行动力学、现代控制、非线性控制、鲁棒控制，再进入高超声速综述、具体控制论文和技术报告。不要一开始只读最新智能控制论文，否则容易把问题写成算法堆叠。

资料类型

类型 用途 阅读重点
飞行动力学教材建立变量和方程基础坐标系、姿态、气动力、配平
现代控制教材建立状态空间和 LQR 基础可控性、稳定性、Riccati 方程
非线性控制教材建立证明能力Lyapunov、反步、ISS、滑模
鲁棒控制教材理解不确定性H 无穷、增益裕度、模型摄动
高超声速综述了解对象难点强非线性、耦合、约束、验证
具体控制论文学习方法细节模型、控制律、证明、指标

检索关键词

英文关键词：hypersonic vehicle control, hypersonic longitudinal dynamics, dynamic pressure constrained control, barrier Lyapunov function hypersonic, control barrier function aerospace, nonlinear dynamic inversion hypersonic, backstepping hypersonic vehicle, sliding mode hypersonic vehicle, integrated guidance and control hypersonic.

中文关键词：高超声速飞行器控制、高超声速纵向动力学、高超声速动压约束、高超声速制导控制一体化、高超声速反步控制、高超声速滑模控制、高超声速自适应控制、高超声速控制屏障函数。

文献记录模板

字段 内容
文献类型教材、综述、方法论文、技术报告
飞行器类型AB-HSV、HGV、再入飞行器等
模型质点、纵向、六自由度、刚柔耦合
控制目标跟踪、镇定、制导、约束、安全、故障
方法PID、LQR、NDI、反步、SMC、BLF、CBF、模型预测控制等
可复现性是否有参数、数据和命令

红旗清单

  • 只展示误差曲线，不报告约束和控制输入。
  • 声称安全，但没有安全集合和不变性证明。
  • 声称鲁棒，但没有不确定性范围。
  • 结论从简化纵向模型外推到完整飞行器。

本附录的目的，是把“读过一篇论文”转成“知道它证明了什么、还没证明什么”。
