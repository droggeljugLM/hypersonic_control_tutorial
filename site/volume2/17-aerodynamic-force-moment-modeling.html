第17章 气动力与气动力矩建模

17.1 问题背景

气动模型是高超声速飞行器控制中连接环境、运动学和控制输入的核心环节。升力、阻力、侧力以及三轴力矩决定轨迹演化、姿态变化、控制效能、约束裕度和不确定性传播。

从控制科学与工程角度看，气动建模至少要回答五个问题：

  • 气动力和力矩在哪个坐标系中表达；
  • 气动系数依赖哪些飞行变量和构型变量；
  • 控制输入如何通过舵面、喷管或其他执行机构影响力和力矩；
  • 稳定导数、阻尼导数和控制导数能否支持局部线性设计；
  • 气动表插值、外推和不确定性如何进入验证与闭环分析。

本章不追求高保真气动理论全覆盖，而是建立后续配平、线化、控制分配、鲁棒控制和六自由度仿真的统一接口。

17.2 气动力和力矩的基本表达

在机体系中，气动力和气动力矩通常写成

FA=q‾S[CXCYCZ],MA=q‾S[c‾Clc‾CmbCn]. \begin{aligned} F_A &= \bar q S \begin{bmatrix} C_X \\ C_Y \\ C_Z \end{bmatrix}, \\ M_A &= \bar q S \begin{bmatrix} \bar c\, C_l \\ \bar c\, C_m \\ b\, C_n \end{bmatrix}. \end{aligned}

其中，FAF_A 表示机体系气动力，MAM_A 表示机体系气动力矩，q‾\bar q 表示动压，SS 表示参考面积，c‾\bar c 表示平均气动弦长，bb 表示展长，CX,CY,CZC_X,C_Y,C_Z 是力系数，Cl,Cm,CnC_l,C_m,C_n 是滚转、俯仰和偏航力矩系数。

这里的符号约定必须在项目中统一。若采用风轴系或稳定轴系，系数表达会改变，但“系数乘以动压和参考尺度”的建模逻辑不变。控制器设计时应始终明确当前使用的坐标系，以及从气动模型到刚体方程的旋转关系。

17.3 气动系数的变量依赖

气动系数通常依赖下列变量：

  • MaMa：Mach 数；
  • hh：高度；
  • α\alpha：攻角；
  • β\beta：侧滑角；
  • ωB=[p,q,r]⊤\omega_B=[p,q,r]^\top：机体系角速度；
  • δ\delta：舵面或等效执行机构输入；
  • ηs\eta_s：弹性模态、热变形或构型参数。

因此，一个较一般的气动系数可写成

C=C(Ma,h,α,β,ωB,δ,ηs). C = C(Ma,h,\alpha,\beta,\omega_B,\delta,\eta_s).

这里的 CC 表示任一气动系数，MaMa、hh、α\alpha、β\beta、ωB\omega_B、δ\delta 和 ηs\eta_s 是它的主要自变量。

其中，CC 可以代表任一力系数或力矩系数。入门模型可以忽略一部分变量，但必须说明被忽略变量将以不确定性、外扰或未建模动态的形式进入后续验证。

17.4 静态项、导数项和控制导数

在局部建模中，气动系数常在某个平衡点附近展开为

C≈C0+CxΔx+CδΔδ+CωΔωB. C \approx C_0 + C_x\,\Delta x + C_\delta\,\Delta \delta + C_\omega\,\Delta \omega_B.

这里的 C0C_0 是静态项，CxC_x、CδC_\delta 和 CωC_\omega 分别对应状态、控制和阻尼方向的局部导数。

其中，C0C_0 表示静态项，Δx\Delta x 表示攻角、侧滑角或其他状态变量的偏差，CxC_x 表示状态导数项，Δδ\Delta \delta 表示控制输入偏差，CδC_\delta 表示控制导数项，ΔωB\Delta \omega_B 表示角速度偏差，CωC_\omega 表示阻尼导数项。

对纵向俯仰力矩，局部线性化常写成

ΔMy≈q‾Sc‾(CmαΔα+Cmqc‾2VΔq+CmδeΔδe⋯), \Delta M_y \approx \bar q S \bar c \left( C_{m_\alpha}\Delta \alpha + C_{m_q}\frac{\bar c}{2V}\Delta q + C_{m_{\delta_e}}\Delta \delta_e \cdots \right),

其中，MyM_y 是俯仰力矩，qq 是俯仰角速度，VV 是速度，δe\delta_e 是升降舵偏角，CmαC_{m_\alpha}、CmqC_{m_q} 和 CmδeC_{m_{\delta_e}} 分别是攻角导数、俯仰阻尼导数和升降舵控制导数。

这里的关键不只是“存在导数”，而是导数决定了输入矩阵的方向、尺度和符号。若控制导数符号写错，闭环设计会从根上失配。

17.5 控制效能矩阵

多执行机构系统常写成

ν=Baδ, \nu = B_a \delta,

这里的 ν\nu 是期望控制量，BaB_a 是控制效能矩阵，δ\delta 是实际舵面向量。

其中，ν\nu 是期望广义控制量或期望力矩向量，BaB_a 是控制效能矩阵，δ\delta 是实际舵面或执行机构偏角向量。

在高超声速飞行器上，BaB_a 可能随 MaMa、hh、α\alpha、β\beta 和动压显著变化。控制分配和动态逆设计不能只在标称点看 BaB_a，还要检查以下问题：

  • 控制效能是否随飞行条件变号；
  • 低动压下是否接近通道失效；
  • 舵面之间是否存在强耦合；
  • 饱和后剩余控制权限是否足够；
  • 故障后 BaB_a 的秩是否下降。

若 BaB_a 不可信，控制分配优化即使数学上可解，也不代表真实执行机构能产生所需力矩。

17.6 气动表与插值

工程中常使用多维气动表描述系数随变量变化的关系，例如

C=𝒯(Ma,h,α,β,δ,ωB,ηs), C = \mathcal{T}(Ma,h,\alpha,\beta,\delta,\omega_B,\eta_s),

其中，𝒯\mathcal{T} 表示气动表或代理模型。多维气动表可能对 δ\delta 呈现强非线性，对 α\alpha 和 β\beta 呈现局部非单调，对 MaMa 和 hh 呈现分段特征。

因此，教程和论文中必须明确：

  • 使用的是全局查表、局部线化还是代理模型；
  • 解耦矩阵是否可能奇异；
  • 控制输入是否超出表格范围；
  • 表格外推是否记录日志并触发保护；
  • 模型误差如何由鲁棒项、观测器或增量式控制补偿。

如果直接把线性控制效能矩阵当作全局模型使用，结果通常只在很窄的包线内成立。

17.7 气动导数与动压效应

考虑纵向教学模型，气动力矩在局部可近似为

ΔMy≈q‾Sc‾(CmαΔα+Cmqc‾2VΔq+CmδeΔδe). \Delta M_y \approx \bar q S \bar c \left( C_{m_\alpha}\Delta \alpha + C_{m_q}\frac{\bar c}{2V}\Delta q + C_{m_{\delta_e}}\Delta \delta_e \right).

这里的 ΔMy\Delta M_y 表示局部俯仰力矩增量，q‾\bar q 通过动压把导数项放大到实际力矩尺度。

其中，CmαC_{m_\alpha} 是攻角导数，CmqC_{m_q} 是俯仰阻尼导数，CmδeC_{m_{\delta_e}} 是升降舵控制导数。

这个表达直接告诉我们：q‾\bar q 是一个放大因子。

  • 低动压时，∂My/∂δe\partial M_y/\partial \delta_e 变小，俯仰控制效能不足；
  • 高动压时，舵面效能增强，但结构载荷、热风险和舵面力矩约束更紧；
  • 动压接近边界时，控制器可能需要降低参考、限制输入或引入效能折减模型。

因此，气动导数不能只在一个标称点报告。至少应随 Ma,h,α,q‾Ma,h,\alpha,\bar q 检查控制效能范围。如果 CmδeC_{m_{\delta_e}} 或 q‾Sc‾Cmδe\bar q S \bar c C_{m_{\delta_e}} 接近零，动态逆和控制分配都可能变得病态。

17.8 气动接口契约

一个可复用的气动模型应明确输入、输出、单位和状态标志。建议最小契约如下：

类别 字段 说明
输入 马赫数 气动表索引和压缩性效应
输入 高度 大气密度和温度相关输入
输入 攻角、侧滑角 气动角，必须注明弧度
输入 机体系角速度 机体系角速度
输入 各舵面实际偏角 各执行机构实际偏角，不是命令偏角
输入 动压 气动力和力矩尺度
输出 机体系气动力 机体系气动力
输出 机体系气动力矩 机体系气动力矩
输出 系数集合 CX,CY,CZ,Cl,Cm,CnC_X,C_Y,C_Z,C_l,C_m,C_n 或等价系数
标志 是否在气动表范围内 表格输入是否落在可用区间内
标志 表格输入是否被限幅 为避免外推而对表格输入进行限幅
标志 uncertainty_sample_id 当前不确定性样本

控制器设计时，气动模型必须使用实际执行机构偏角。如果把命令偏角直接送入气动模型，就会掩盖执行机构滞后、饱和、速率限制和故障带来的真实力矩损失。

17.9 气动数据验收矩阵

教学合成气动表也需要验收。一个最小检查矩阵可写为：

检查 扫描变量 通过口径
攻角趋势 α\alpha 升力趋势、阻力趋势和俯仰力矩趋势符合设定符号
舵面效能 δe,δa,δr\delta_e,\delta_a,\delta_r 力矩方向和控制器约定一致
侧滑趋势 β\beta 侧力、滚转和偏航趋势符号一致
角速度阻尼 p,q,rp,q,r 动导数提供合理阻尼或明确说明简化
动压尺度 q‾\bar q 力和力矩按动压线性缩放
表格边界 Ma,α,β,δMa,\alpha,\beta,\delta 超界时有标志、限幅或失败记录
导数平滑性 差分步长扫描 局部线化导数不过度跳变
不确定性 系数扰动样本 扰动幅值、种子和作用通道可追溯

这张矩阵的作用不是证明气动模型真实，而是证明当前模型在教学范围内自洽。若要支撑高保真结论，还需要风洞、CFD、飞行数据或经过审查的气动数据库。

17.11 常见错误

  1. 升力、阻力、侧力坐标系没有说明。
  2. 攻角单位在度和弧度之间混用。
  3. 气动表外推没有任何日志字段。
  4. 舵面正方向和控制器正方向相反。
  5. 用静态气动表，却声称验证了高速姿态动态。
  6. 线化导数由噪声表格差分得到，却不检查平滑性。
  7. 气动不确定性只在文字中出现，没有进入仿真。
  8. 合成教学气动表被表述成高保真数据库。
  9. 用命令舵偏计算气动力，而不是用执行机构实际舵偏。
  10. 不检查低动压下控制效能矩阵是否病态。

17.12 本章小结

气动力与气动力矩建模决定高超声速控制器是否有真实对象可控。参考书级教程必须把气动系数、气动表、稳定导数、控制导数、插值外推、不确定性和验证指标讲清楚。后续配平、线化、增益调度、控制分配、鲁棒控制、NDI/INDI 和六自由度仿真都建立在本章接口之上。
